## Overview

The ingestion system is the entry point for all data into Dalgo Lite. It must be:

- **Simple enough** for M&E officers who've never used a data platform
- **Powerful enough** to handle multiple sources, schedules, and schema changes
- **Reliable enough** to sync data automatically without manual intervention

### Core Principles

| Principle | Implementation |
|-----------|----------------|
| **Zero jargon** | "Connect your data" not "Configure ingestion pipeline" |
| **Visual feedback** | Show data previews at every step |
| **Guided setup** | Step-by-step wizards, not forms |
| **Forgiveness** | Easy to undo, change, or reconnect |
| **Transparency** | Show what's happening without overwhelming |

## Part B: UX Review for Non-Technical Users

### Target User Profile

**Primary User:** M&E Officer at an NGO
- **Technical skills:** Advanced Excel, basic Google Sheets formulas
- **Data understanding:** Knows their data well, understands rows/columns/filtering
- **Pain points:** Spends hours manually copying data between sheets
- **Doesn't know:** SQL, APIs, databases, ETL, schemas, pipelines

### UX Principles Applied

#### 1. Familiar Terminology

| Technical Term | We Use Instead | Why |
|----------------|----------------|-----|
| Database/Warehouse | "Your database" or "Where your data is stored" | Focus on ownership |
| Source/Destination | "From" â†’ "To" with arrows | Visual metaphor |
| Sync | "Update" or "Check for changes" | Action-oriented |
| Schema | "Structure" or "columns" | Excel language |
| Pipeline | "Connection" | Simple relationship |
| Credentials | "Login details" or "Account" | Familiar concept |
| API Token | "Access key" or "Secret code" | Less technical |
| Incremental | "Only new data" | Outcome-focused |

#### 2. Progressive Disclosure

**Level 1 (Default):** Status at a glance
- Green dot = working
- Row counts
- Last sync time

**Level 2 (Click for more):** Source settings
- Sync schedule
- Recent history
- Connection details

**Level 3 (Advanced):** Technical details
- Raw logs
- Schema information
- Manual SQL access

#### 3. Error Messages

| Bad (Technical) | Good (Human) |
|-----------------|--------------|
| "401 Unauthorized" | "Please reconnect your Google account" |
| "Connection timeout" | "We couldn't reach Google Sheets. Please try again." |
| "Schema mismatch" | "Your spreadsheet structure changed. [Review changes]" |
| "Rate limited" | "We're checking too often. Will try again in 5 minutes." |
| "Null constraint violation" | "Some required fields are empty in your data" |

#### 4. Guided Workflows

Every complex action uses a wizard pattern:
1. **Clear title** - What are we doing?
2. **Context** - Why does this matter?
3. **Single choice per screen** - Reduce cognitive load
4. **Preview before commit** - Build confidence
5. **Clear next step** - Always know what's coming

#### 5. Visual Feedback

| State | Visual Indicator | User Understanding |
|-------|------------------|-------------------|
| Connected | Green dot â— | "All good" |
| Syncing | Blue spinner â†» | "Working on it" |
| Needs attention | Yellow warning âš  | "Check this" |
| Error | Red alert âŒ | "Something's wrong" |
| Not set up | Gray circle â—‹ | "Not done yet" |

### UX Checklist

| Principle | Implementation | Status |
|-----------|----------------|--------|
| No jargon | All technical terms replaced | âœ… |

### User Testing Scenarios

1. **First-time setup:** Can a user connect their first Google Sheet in under 5 minutes?
2. **Add Kobo:** Can a user figure out where to find their API token?
3. **Fix error:** When sync fails, can user understand what to do?
4. **Schema change:** When columns change, can user review and accept?
5. **Check status:** At a glance, can user tell if everything is working?

# backend/app/connectors/base.py

from abc import ABC, abstractmethod
from typing import Any, Dict, List, Optional
from pydantic import BaseModel
import dlt

class ConnectorConfig(BaseModel):
    """Base configuration for all connectors."""
    pass

class ConnectorResource(BaseModel):
    """Represents a syncable resource (form, sheet, table)."""
    id: str
    name: str
    description: Optional[str] = None
    row_count: Optional[int] = None
    last_modified: Optional[str] = None

class SyncResult(BaseModel):
    """Result of a sync operation."""
    success: bool
    rows_synced: int
    load_id: str
    error: Optional[str] = None
    schema_changes: Optional[Dict] = None

class BaseConnector(ABC):
    """
    Base class for all data source connectors.

    Each connector must implement:
    1. get_config_schema() - JSON Schema for configuration form
    2. test_connection() - Validate credentials
    3. list_resources() - List available forms/sheets/tables
    4. create_dlt_source() - Create DLT source for syncing
    """

    connector_type: str  # e.g., "google_sheets", "kobo", "csv"
    display_name: str    # e.g., "Google Sheets", "KoboToolbox"
    description: str     # Short description for UI
    icon: str           # Icon identifier

    @abstractmethod
    def get_config_schema(self) -> Dict[str, Any]:
        """
        Return JSON Schema for connector configuration.
        Used to generate the configuration form in the UI.
        """
        pass

    @abstractmethod
    async def test_connection(self, config: Dict[str, Any]) -> bool:
        """
        Test if the provided credentials are valid.
        Returns True if connection successful, raises exception otherwise.
        """
        pass

    @abstractmethod
    async def list_resources(
        self,
        config: Dict[str, Any]
    ) -> List[ConnectorResource]:
        """
        List available resources (forms, sheets, tables) that can be synced.
        """
        pass

    @abstractmethod
    def create_dlt_source(
        self,
        config: Dict[str, Any],
        resources: List[str],
        incremental: bool = True
    ) -> dlt.Source:
        """
        Create a DLT source for the selected resources.
        """
        pass

    async def sync(
        self,
        config: Dict[str, Any],
        resources: List[str],
        destination_url: str,
        schema_name: str
    ) -> SyncResult:
        """
        Execute a sync operation.
        """
        source = self.create_dlt_source(config, resources)

        pipeline = dlt.pipeline(
            pipeline_name=f"{self.connector_type}_{schema_name}",
            destination=dlt.destinations.postgres(destination_url),
            dataset_name=schema_name
        )

        load_info = pipeline.run(source)

        # Extract schema changes
        schema_changes = {}
        for package in load_info.load_packages:
            if package.schema_update:
                schema_changes.update(package.schema_update)

        return SyncResult(
            success=not load_info.has_failed_jobs,
            rows_synced=load_info.metrics.get("rows_total", 0),
            load_id=load_info.load_id,
            schema_changes=schema_changes if schema_changes else None
        )

        config: Dict[str, Any],
        resources: List[str],
        incremental: bool = True
    ) -> dlt.Source:
        base_url = self._get_base_url(config)

        return rest_api_source({
            "client": {
                "base_url": f"{base_url}/api/v2/",
                "auth": {
                    "type": "bearer",
                    "token": config["api_token"]
                }
            },
            "resources": [
                {
                    "name": f"kobo_{form_id}",
                    "endpoint": {
                        "path": f"assets/{form_id}/data/",
                        "paginator": {
                            "type": "offset",
                            "limit": 100,
                            "offset_param": "start",
                            "limit_param": "limit"
                        },
                        "incremental": {
                            "cursor_path": "_submission_time",
                            "initial_value": "2020-01-01T00:00:00"
                        } if incremental else None
                    }
                }
                for form_id in resources
            ]
        })

    def _get_base_url(self, config: Dict[str, Any]) -> str:
        if config["server"] == "custom":
            return config["custom_url"].rstrip("/")
        return self.SERVERS[config["server"]]

#### 3.1 Schema Tracking Service

```python
# backend/app/services/schema_tracker.py

from typing import Dict, List, Optional
from datetime import datetime
from pydantic import BaseModel
import dlt

class SchemaChange(BaseModel):
    """Represents a single schema change."""
    source_id: str
    change_type: str  # 'column_added', 'column_removed', 'type_changed'
    column_name: str
    table_name: str
    old_value: Optional[Dict] = None
    new_value: Optional[Dict] = None
    detected_at: datetime = datetime.utcnow()

class SchemaSnapshot(BaseModel):
    """Snapshot of table schema at a point in time."""
    source_id: str
    version: int
    version_hash: str
    columns: Dict[str, Dict]  # column_name -> {type, nullable, etc.}
    captured_at: datetime

class SchemaTracker:
    """
    Tracks schema changes across syncs.

    Uses DLT's built-in schema tracking plus our own layer for:
    - User-friendly change descriptions
    - Impact analysis (which transforms are affected)
    - Change history
    """

    def __init__(self, db_session):
        self.db = db_session

    async def capture_schema(
        self,
        source_id: str,
        pipeline: dlt.Pipeline
    ) -> SchemaSnapshot:
        """Capture current schema after a sync."""
        schema = pipeline.default_schema

        # Extract column info for data tables only
        columns = {}
        for table_name, table in schema.tables.items():
            if not table_name.startswith("_dlt_"):
                columns[table_name] = {
                    col_name: {
                        "data_type": col.get("data_type"),
                        "nullable": col.get("nullable", True),
                        "primary_key": col.get("primary_key", False)
                    }
                    for col_name, col in table.get("columns", {}).items()
                }

        snapshot = SchemaSnapshot(
            source_id=source_id,
            version=schema.version,
            version_hash=schema.version_hash,
            columns=columns,
            captured_at=datetime.utcnow()
        )

        await self._save_snapshot(snapshot)
        return snapshot

    async def detect_changes(
        self,
        source_id: str,
        load_info: dlt.LoadInfo
    ) -> List[SchemaChange]:
        """
        Detect schema changes from DLT load_info.

        DLT provides schema_update in load_info.load_packages,
        which contains all columns that were added or modified.
        """
        changes = []
        previous = await self._get_latest_snapshot(source_id)

        for package in load_info.load_packages:
            if not package.schema_update:
                continue

            for table_name, table_info in package.schema_update.items():
                if table_name.startswith("_dlt_"):
                    continue

                for col_name, col_info in table_info.get("columns", {}).items():
                    # Check if this is a new column or type change
                    prev_col = None
                    if previous and table_name in previous.columns:
                        prev_col = previous.columns[table_name].get(col_name)

                    if prev_col is None:
                        # New column
                        changes.append(SchemaChange(
                            source_id=source_id,
                            change_type="column_added",
                            column_name=col_name,
                            table_name=table_name,
                            new_value=col_info
                        ))
                    elif prev_col.get("data_type") != col_info.get("data_type"):
                        # Type changed
                        changes.append(SchemaChange(
                            source_id=source_id,
                            change_type="type_changed",
                            column_name=col_name,
                            table_name=table_name,
                            old_value=prev_col,
                            new_value=col_info
                        ))

        # Check for removed columns
        if previous:
            current_schema = await self._get_current_columns(source_id, load_info)

            for table_name, prev_columns in previous.columns.items():
                curr_columns = current_schema.get(table_name, {})

                for col_name in prev_columns:
                    if col_name not in curr_columns:
                        changes.append(SchemaChange(
                            source_id=source_id,
                            change_type="column_removed",
                            column_name=col_name,
                            table_name=table_name,
                            old_value=prev_columns[col_name]
                        ))

        # Save changes to database
        if changes:
            await self._save_changes(changes)

        return changes

    async def get_impact_analysis(
        self,
        changes: List[SchemaChange]
    ) -> Dict[str, List[str]]:
        """
        Analyze which transformations are affected by schema changes.
        Returns mapping of change -> list of affected transformation names.
        """
        impact = {}

        for change in changes:
            # Query transformations that reference this column
            affected = await self.db.execute("""
                SELECT t.id, t.name
                FROM transformations t
                WHERE t.recipe::text LIKE :pattern
            """, {"pattern": f'%"{change.column_name}"%'})

            if affected:
                impact[f"{change.table_name}.{change.column_name}"] = [
                    row.name for row in affected
                ]

        return impact

    async def _save_snapshot(self, snapshot: SchemaSnapshot):
        """Save schema snapshot to database."""
        await self.db.execute("""
            INSERT INTO source_schema_history
            (source_id, schema_version, schema_hash, columns, captured_at)
            VALUES (:source_id, :version, :hash, :columns, :captured_at)
        """, {
            "source_id": snapshot.source_id,
            "version": snapshot.version,
            "hash": snapshot.version_hash,
            "columns": snapshot.columns,
            "captured_at": snapshot.captured_at
        })

    async def _get_latest_snapshot(
        self,
        source_id: str
    ) -> Optional[SchemaSnapshot]:
        """Get the most recent schema snapshot for a source."""
        result = await self.db.execute("""
            SELECT * FROM source_schema_history
            WHERE source_id = :source_id
            ORDER BY captured_at DESC
            LIMIT 1
        """, {"source_id": source_id})

        if result:
            return SchemaSnapshot(**result)
        return None

    async def _save_changes(self, changes: List[SchemaChange]):
        """Save schema changes for user review."""
        for change in changes:
            await self.db.execute("""
                INSERT INTO schema_changes
                (source_id, change_type, column_name, table_name,
                 old_value, new_value, detected_at, status)
                VALUES
                (:source_id, :change_type, :column_name, :table_name,
                 :old_value, :new_value, :detected_at, 'pending')
            """, change.dict())
```

#### 3.2 Sync Service (Orchestration)

```python
# backend/app/services/sync_service.py

from typing import List, Optional
from datetime import datetime
import asyncio
from ..connectors.registry import ConnectorRegistry
from .schema_tracker import SchemaTracker

class SyncService:
    """
    Orchestrates data synchronization for all sources.

    Responsibilities:
    - Run syncs on schedule
    - Track sync status and history
    - Detect and report schema changes
    - Handle errors gracefully
    """

    def __init__(self, db_session, notification_service):
        self.db = db_session
        self.schema_tracker = SchemaTracker(db_session)
        self.notifications = notification_service

    async def sync_source(self, source_id: str) -> SyncResult:
        """
        Sync a single source.

        Steps:
        1. Get source configuration
        2. Create DLT source via connector
        3. Run DLT pipeline
        4. Detect schema changes
        5. Update sync history
        6. Notify user if needed
        """
        # Get source config
        source = await self._get_source(source_id)
        if not source:
            raise ValueError(f"Source not found: {source_id}")

        # Update status to syncing
        await self._update_status(source_id, "syncing")

        try:
            # Get connector
            connector = ConnectorRegistry.get(source.connector_type)

            # Get organization's warehouse URL
            org = await self._get_organization(source.organization_id)
            warehouse_url = org.warehouse_url
            schema_name = f"org_{org.id}"

            # Run sync
            result = await connector.sync(
                config=source.config,
                resources=source.selected_resources,
                destination_url=warehouse_url,
                schema_name=schema_name
            )

            # Detect schema changes
            if result.schema_changes:
                changes = await self.schema_tracker.detect_changes(
                    source_id,
                    result.schema_changes
                )

                if changes:
                    # Get impact analysis
                    impact = await self.schema_tracker.get_impact_analysis(changes)

                    # Notify user
                    await self.notifications.send_schema_change_alert(
                        source_id=source_id,
                        changes=changes,
                        impact=impact
                    )

            # Update source record
            await self._update_source_after_sync(
                source_id=source_id,
                rows_synced=result.rows_synced,
                success=result.success
            )

            # Save sync history
            await self._save_sync_history(
                source_id=source_id,
                result=result,
                schema_changes=result.schema_changes
            )

            return result

        except Exception as e:
            # Update status to error
            await self._update_status(source_id, "error", str(e))

            # Notify user of failure
            await self.notifications.send_sync_error_alert(
                source_id=source_id,
                error=str(e)
            )

            raise

    async def sync_all_sources(self, organization_id: str) -> List[SyncResult]:
        """Sync all sources for an organization."""
        sources = await self._get_org_sources(organization_id)

        # Sync in parallel (with concurrency limit)
        semaphore = asyncio.Semaphore(3)  # Max 3 concurrent syncs

        async def sync_with_limit(source):
            async with semaphore:
                return await self.sync_source(source.id)

        results = await asyncio.gather(
            *[sync_with_limit(source) for source in sources],
            return_exceptions=True
        )

        return results

    async def schedule_sync(
        self,
        source_id: str,
        frequency: str,
        time_of_day: Optional[str] = None
    ):
        """
        Schedule automatic syncs for a source.

        Frequencies: '15min', 'hourly', 'daily', 'manual'
        """
        await self.db.execute("""
            UPDATE sources
            SET sync_frequency = :frequency,
                sync_time = :time_of_day,
                next_sync_at = :next_sync
            WHERE id = :source_id
        """, {
            "source_id": source_id,
            "frequency": frequency,
            "time_of_day": time_of_day,
            "next_sync": self._calculate_next_sync(frequency, time_of_day)
        })
```
